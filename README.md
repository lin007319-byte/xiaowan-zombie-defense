## v4.3.0 · 蘑菇毒气攻击

- 大喷菇每 1.45 秒向前喷出一整行毒气，同行目标各受到 24 点伤害。
- 忧郁菇保留九宫格三连喷和 1.25 秒休息节奏，每一喷造成 28 点伤害，攻击表现改为范围毒气。
- 被毒气命中的僵尸会显示“中毒”状态，并配有独立的毒雾扩散特效。

## v4.2.2 · 直线轨道锁定

- 每颗直射子弹出生时记录固定水平轨道，飞行全过程强制保持同一高度。
- 穿透、点燃和裂荚子弹反弹后仍锁定原轨道，不会产生任何上下漂移。
- 只有杨桃等明确的多方向范围子弹允许沿斜线飞行。

## v4.2.1 · 连射弹道优化

- 双发、裂荚、机枪及其他多发植物的同方向子弹改为同一条水平直线。
- 多颗子弹仅在发射位置前后错开，不再通过角度、速度或高度差形成散射。
- 裂荚射手向后的子弹仍向后飞行，但保持水平直线；杨桃和忧郁菇等范围攻击不受影响。

## v4.2 · 等价材料融合与基础植物扩充

- 融合判定会把配方中的重复基础植物识别为对应进阶植物；例如“豌豆射手×4”与“机枪射手”等价。
- 樱桃机枪射手支持“机枪射手＋樱桃炸弹”，并保留樱桃裂荚射手继续加入豌豆等逐级路径。
- 等价规则覆盖全部可拆解配方，目前共有 168 个可执行融合步骤。
- 基础卡扩充为 48 种，新增胆小菇、三线射手、窝瓜、缠绕水草、猫尾草、云杉弓手、水滴芦荟等文件配方所需植物；可从基础卡逐级制作 142 种融合植物。

## v4.1 · 融合能力实装

- 修复 200 植物替换后仍按旧编号触发能力的问题，改为每株植物独立能力数据。
- 双发射手每轮 2 发、裂荚射手每轮 3 发并向后发射后反弹、机枪射手每轮 4 发。
- 图鉴说明中的穿透、溅射、击退、寒冰、火焰、黄油定身、范围、治疗和命中产阳光会自动映射为战斗效果。
- 豌豆向日葵恢复每第三发命中掉落 5 阳光，并按 25 秒生产 25 阳光。

## v4.0 · 200 种新文件植物

## 运行

直接打开 `index.html`，或在目录中启动任意静态服务器：

```bash
python3 -m http.server 4173
```

浏览器访问 `http://127.0.0.1:4173/`。

推荐使用最新版 Chrome、Edge 或 Safari。游戏可离线运行，不会上传数据。

## 操作

- 点击上方植物卡，再点击草坪空格种植。
- 所有植物都能拖动：拖到空格移动，拖到另一株植物上完成融合。
- 相同基础植物互相融合会升星，最高三星。
- 点击掉落的阳光收集资源。
- 空格键暂停。
- 按住 F3 开启时停，战斗降为 8% 流速；松开恢复。部分笔记本需要按 `Fn + F3`。

- 无限波次防守：每 30 秒进入下一波，怪物血量与速度持续成长；出怪采用较舒缓的休闲节奏，没有通关终点。
- 每一路推土机拥有两次救场机会，第二次触发后才会彻底耗尽；同一路再次被突破才判定失败。
- 每波提供阳光奖励，每十波可为一台受损推土机补充一次机会，最高恢复到两次；失败时统计生存波数、击杀与融合次数。

- 两张透明角色图集统一加载，减少绘制状态切换和文件请求，保留程序化角色作为图片尚未载入时的后备显示。

## 无限模式平衡

- 初始阳光提升至 400，阳光停留一段时间后会自动拾取。
- 出怪间隔再次整体延长：前十波约 9.8 秒逐步降至 4.8 秒，极限间隔不低于 2.7 秒；场上活跃僵尸软上限降至 48，第十波后的刷新加速更平缓。
- 每局开场提供完整 10 秒无怪布阵保护期，倒计时结束后才生成第一只僵尸。
- 前十波逐步引入全部怪物，十一波后进入循环精英尸潮并持续成长。
- 基础植物耐久按融合版常见的 300 点标尺统一，高坚果为 8000；普僵采用 270 韧性、50/0.5 秒啃咬，巨人采用 3000 韧性，其余防具僵尸按本体与护甲结构折算。
- 默认标记为“休闲”难度，目标是第一次游玩也能形成完整五路防线。
