## 运行

直接打开 `index.html`，或在目录中启动任意静态服务器：

```bash
python3 -m http.server 4173
```

浏览器访问 `http://127.0.0.1:4173/`。

推荐使用最新版 Chrome、Edge 或 Safari。游戏可离线运行，不会上传数据。

## 操作

- 点击下方植物卡，再点击草坪空格种植。
- 所有植物都能拖动：拖到空格移动，拖到另一株植物上完成定向融合。
- 相同基础植物互相融合会升星，最高三星。
- 点击掉落的阳光收集资源。
- 空格键暂停。
- 按住 F3 开启时停，战斗降为 8% 流速；松开恢复。部分笔记本需要按 `Fn + F3`。

## v2.4 内容范围

- 25 株基础植物、625 个定向组合全部有结果。
- 新增大嘴花、火炬树桩、高坚果、三叶草和地刺：分别提供吞噬、子弹点燃、高墙阻跳、吹落飞行单位与地面持续伤害。
- F3 按住式时停会把整场战斗减速到 8%，UI 与帧率统计仍保持正常刷新。
- 新增毁灭菇、寒冰菇、杨桃星炮、路灯花、磁力菇；分别提供大范围爆破、全场冻结、五向弹道、飞行照明和金属剥离能力。
- 在原有 5 种植物上新增玉米投手、火爆辣椒、星光菇、仙人掌、大蒜、咖啡豆、西瓜投手、竹筒连弩、治疗莲花和南瓜堡垒。
- 16 类僵尸；第 11 波开始加入撑杆僵尸、飞翼僵尸与气球僵尸。撑杆僵尸能越过第一株植物，飞行单位会绕过普通阻挡。
- 巨人僵尸拥有蓄力、举槌、砸地和冲击波动作；普通僵尸增加挥臂、前扑和啃咬弧线，植物增加射击后坐、吞噬前探、产能绽放与爆发蓄力。
- 地面僵尸不再以静态贴图悬浮平移：左右腿交替迈步和抬脚、双臂反向摆动、躯干随落脚轻微倾斜，脚下阴影固定在地面。
- 子弹按植物类型采用星形、冰晶、光弹、玉米、西瓜等不同造型，并增加光轨、旋转、命中碎屑、爆炸环与全场冰封效果。
- 无限波次防守：每 30 秒进入下一波，怪物血量与速度持续成长；出怪采用较舒缓的休闲节奏，没有通关终点。
- 每一路推土机拥有两次救场机会，第二次触发后才会彻底耗尽；同一路再次被突破才判定失败。
- 每波提供阳光奖励，每十波可为一台受损推土机补充一次机会，最高恢复到两次；失败时统计生存波数、击杀与融合次数。
- 25 株植物与 16 类僵尸均有手绘质感角色插画或专属绘制：植物拥有明确的根茎、叶片、花冠或果实结构；新增角色采用原创融合风格重绘，不直接复制第三方游戏贴图。
- 植物卡片同步使用角色肖像；场上保留呼吸浮动、行走摇摆、受击闪白、火焰、冰霜、眩晕、护盾、星级和融合基因特效。
- 两张透明角色图集统一加载，减少绘制状态切换和文件请求，保留程序化角色作为图片尚未载入时的后备显示。

## 无限模式平衡

- 初始阳光提升至 400，阳光停留一段时间后会自动拾取。
- 出怪间隔再次整体延长：前十波约 9.8 秒逐步降至 4.8 秒，极限间隔不低于 2.7 秒；场上活跃僵尸软上限降至 48，第十波后的刷新加速更平缓。
- 前十波逐步引入全部怪物，十一波后进入循环精英尸潮并持续成长。
- 基础植物耐久按融合版常见的 300 点标尺统一，高坚果为 8000；普僵采用 270 韧性、50/0.5 秒啃咬，巨人采用 3000 韧性，其余防具僵尸按本体与护甲结构折算。
- 默认标记为“休闲”难度，目标是第一次游玩也能形成完整五路防线。
